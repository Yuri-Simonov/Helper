Что такое "Protractor"?

Protractor - фреймворк комплексного тестирования, с помощью которого можно тестировать Angular приложение прямо в браузере, имитируя взаимодействия, как c реальным пользователем.

По умолчанию с ним в коробке идет 2 фреймворка для тестирования:

  • Юнит тесты на Jasmine и Karma
  • Комплексные тесты на Protractor

Основное различие - первый используется для тестирования логики компонентов и сервисов, а второй используется для проверки функциональности высокого уровня приложения (в том числе и UI элементы).

В каком файле нужно подключать сторонние скрипты и стили и почему?

Cторонние скрипты и стили должны подключаться в angular.json, а не в index.html как это делается в верстке, например.

Почему именно так? Важно, чтобы все настройки были в одном месте, а не размазаны по всему приложению. Причем файл angular.json как раз для этого и предназначен, т.к. является главным конфигурационным файлом Angular Workspace.

Зачем нужно свойство "bootstrap" в декораторе "@NgModule()"?

В свойстве bootstrap указывается компонент, который будет использоваться в качестве основного при загрузке модуля.

Что такое "Angular CLI"?

Angular CLI - это npm-модуль, реализующий интерфейс командной строки для создания, разработки и поддержки Angular приложений. В системе он должен быть установлен глобально.

Что такое модуль в Angular?

Angular модуль - это класс c декоратором @NgModule(), который служит изолирующей логической объединяющей структурой для компонентов, директив, фильтров и сервисов. 

@NgModule({{'{'}}
    declarations: [ParentComponent], // дочерние компоненты, директивы и фильтры
    imports: [SomeModule], // другие модули, используемые в текущем модуле
    providers: [SomeService], // импортирование в модуль необходимых сервисов приложения
    bootstrap: [ParentComponent], // имя главного компонента модуля
    exports: [SomeComponent], // экспортируемые компоненты, директивы и фильтры, которые могут быть использованы другими модулями
{{'}'}})
export class ParentModule {{'{}'}} // класс, к которому привязывается декоратор @NgModule()
Что такое компонент в Angular?

Angular компонент - это часть интерфейса приложения с собственной логикой. Все компоненты в совокупности и есть Angular App.

За создание компонента отвечает декоратор @Component(). 

@Component{{'({'}}			
    selector: 'app-root', // название компонента
    templateUrl: './app.component.html', // путь к HTML-файлу. Является обязательным свойством
    styleUrls: ['./app.component.scss'], // массив путей к SCSS-файлам
{{'})'}}
export class AppComponent {{'{}'}} // класс, к которому привязывается декоратор @Component()
